tellers do name it, they name it **differently** (unknown vs angel). A consistent self-identification would
count as a datum; an identity that shifts by narrator reads as legend-variance. The residue test (verifiable
information excluded from ordinary means) returns null throughout. Untestable and unsupported, H-3 stays **set
aside at null weight — which does not equal disproven.** Taylor never speaks on record; the silence holds no
space open for belief, exactly as the unreleased Varginha autopsy videos held none.
