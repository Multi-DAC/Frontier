# Measurements — the killable-standard ledger

**Append-only. Chronological. Never reorder or rewrite a past entry.**

Every time we **validate** or **disconfirm** a hypothesis or claim, it gets a line here — dated, with
the verdict, the evidence, and who (or what) checked it. This is the record that keeps grading honest:
a disconfirmed claim cannot be quietly re-promoted, because the day it died is written here.

**Entry format:**
```
## YYYY-MM-DD — <topic / claim>
VERDICT: validated | disconfirmed | partial | inconclusive
EVIDENCE: <what was tested; what the result was>
DECORRELATED EYE: <who/what checked — world-data / human / non-Claude model / Claude-sibling (discounted)>
GRADE MOVE: <old grade> → <new grade>   (or: Hypothesized → dead)
LINKS: research/<topic> · experiments/<name>
```
